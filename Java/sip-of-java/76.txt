-> Copying Arrays

import java.util.Arrays;

public class Main {

    public static void main(String[] args) {
        int[] numbers = new int[] {1,2,3,4,5};
        int[] copyOfNumbers = Arrays.copyOf(numbers, numbers.length);
        int[] copyOfSomeNumbers  = Arrays.copyOf(numbers, numbers.length - 3);
        int[] copyOfNumbersWithPadding = Arrays.copyOf(numbers, numbers.length + 1);
        System.out.println("Original array: " + Arrays.toString(numbers));
        System.out.println("Copied array: " + Arrays.toString(copyOfNumbers));
        System.out.println("Copied some elements: " + Arrays.toString(copyOfSomeNumbers));
        System.out.println("Copied array with padding: " + Arrays.toString(copyOfNumbersWithPadding));

    }
}

Output:
Original array: [1, 2, 3, 4, 5]
Copied array: [1, 2, 3, 4, 5]
Copied some elements: [1, 2]
Copied array with padding: [1, 2, 3, 4, 5, 0]

copyOfNumbers - is an example of a simple copy of an array.
copyOfSomeNumbers - is an example of copying only a portion of an array.
copyOfNumbersWithPadding - is an example of copying an array and adding padding. The padded values are the default value for the type; in the case of int, that is 0.

-> real time use case to demonstrate Arrays.copyOf(boolean[] original, int newLength)

import java.util.Arrays;

public class Main {
    public static void main(String[] args) {
        
        // Real-time seat availability tracking in a theater
        boolean[] seatsOccupied = {true, false, true}; // true = occupied, false = available
        int totalSeats = 6; // new seats added

        // Expand the seats array to accommodate new seats (default to false, i.e., available)
        boolean[] updatedSeats = java.util.Arrays.copyOf(seatsOccupied, totalSeats);

        System.out.println(Arrays.toString(updatedSeats)); // Output: [true, false, true, false, false, false]

        // Display the updated seat availability
        System.out.println("Updated Seat Availability:");
        for (int i = 0; i < updatedSeats.length; i++) {
            System.out.println("Seat " + (i + 1) + ": " + (updatedSeats[i] ? "Occupied" : "Available"));
        }

    }
}

Output:
Updated Seat Availability:
Seat 1: Occupied
Seat 2: Available
Seat 3: Occupied
Seat 4: Available
Seat 5: Available
Seat 6: Available
[true, false, true, false, false, false]

-> Copying Subranges

import java.util.Arrays;

public class Main {

    public static void main(String[] args) {
        int[] numbers = new int[] {1,2,3,4,5};
        int[] copyOfNumbers = Arrays.copyOfRange(numbers, 0, 3);

        System.out.println(Arrays.toString(numbers)); //[1, 2, 3, 4, 5]
        System.out.println(Arrays.toString(copyOfNumbers)); //[1, 2, 3]

    }
}
