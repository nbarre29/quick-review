### 3. **XOR Mention (Slight Clarification)**

You mentioned:
> Swapping using XOR (why it works for numbers, not Strings)

**Quick clarification for completeness:**

XOR works for **primitives**:
```java
a = a ^ b;
b = a ^ b;  // Now b = original a
a = a ^ b;  // Now a = original b
```

**Why it doesn't work for Strings:**
- XOR is a **bitwise operator** (works on `int`, `long`, etc.)
- Strings are **objects**, not primitive values
- `^` operator is not defined for String type
- Would get compilation error: `operator ^ cannot be applied to java.lang.String`
